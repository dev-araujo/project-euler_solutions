# Project Euler Solutions

![Project Euler](assets/logo-euler.jpg)

This repository contains solutions to the problems from Project Euler, a website that offers a series of challenging mathematical and computational problems.

## 🚀 Getting Started

### Prerequisites

- Node.js (v14 or higher is recommended)

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/araujocode/project-euler_solutions.git
   ```

2. Navigate to the project directory:
   ```bash
   cd project-euler_solutions
   ```

## 💻 Usage

Each problem's solution is organized in its own directory. To run a specific problem's solution, simply execute the corresponding `index.js` file using Node.js:

```bash
node 01_multiple-of-3-or-5/index.js
```
